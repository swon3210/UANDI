# 사진 갤러리 3단계: 업로더 구분 + 선택 모드 + 일괄 이동

> 상위 명세: `03-photo-gallery.md`
> 선행 단계: `03a-gallery-scaffold.md`, `03b-photo-upload-detail.md`

## 범위

- 업로더 아바타 뱃지 (전체/폴더/태그 그리드에 추가)
- 업로더 필터 칩 (전체 / 나 / 연인) — 폴더·태그 상세에 추가
- 선택 모드 + 일괄 폴더 이동 — 폴더 상세에 추가

## UI 변경사항

### 그리드 썸네일에 아바타 뱃지 추가

전체/폴더/태그 그리드의 모든 썸네일 좌하단에 업로더의 아바타 뱃지(24px 원형) 추가.

```
┌─────┐ ┌─────┐
│ 🧑‍  │ │     │   ← 좌하단에 업로더 아바타 뱃지 (작은 원형)
└─────┘ └─────┘
```

```tsx
<Avatar className="absolute bottom-1 left-1 h-6 w-6 border-2 border-white">
  <AvatarImage src={uploaderPhotoURL} />
</Avatar>
```

### 폴더 상세 — 필터 칩 + 슬라이드쇼 버튼 + 선택 모드

#### 기본 모드

```
┌─────────────────────────┐
│  ← 제주도 여행    [▶] [☑] │  ← Header: 폴더명 + 슬라이드쇼 + 선택 모드
├─────────────────────────┤
│  [ 전체 | 나 | 연인 ]    │  ← 업로더 필터 칩
├─────────────────────────┤
│                           │
│  ┌─────┐ ┌─────┐        │
│  │ 🧑‍  │ │     │        │  ← 2열 그리드 + 아바타 뱃지
│  └─────┘ └─────┘        │
│  ...                     │
│                           │
└─────────────────────────┘
```

- 상단 필터 칩: **전체 / 나 / 연인** — 업로더 기준 필터링 (클라이언트 사이드)
- [▶] 슬라이드쇼 버튼 → 현재 필터 기준 사진들로 슬라이드쇼 진입 (4단계에서 연결)
- [☑] 선택 모드 진입 버튼

#### 선택 모드

헤더의 [☑] 버튼 클릭 시 선택 모드로 전환.

```
┌─────────────────────────┐
│  ✕ 3장 선택          [이동] │  ← 닫기 + 선택 수 + 이동 버튼
├─────────────────────────┤
│                           │
│  ┌─────┐ ┌─────┐        │
│  │ ✓  │ │ ✓  │        │  ← 선택된 사진에 체크 오버레이
│  └─────┘ └─────┘        │
│  ┌─────┐ ┌─────┐        │
│  │     │ │ ✓  │        │
│  └─────┘ └─────┘        │
│  ...                     │
│                           │
└─────────────────────────┘
```

- 사진 탭 시 선택/해제 토글 (체크 오버레이 표시)
- 헤더에 선택된 사진 수 표시 ("3장 선택")
- **[이동]** 버튼: 1장 이상 선택 시 활성화 → 폴더 선택 Bottom Sheet 표시
- ✕ 버튼: 선택 모드 해제, 기본 모드로 복귀

#### 폴더 이동 Bottom Sheet (선택 모드에서 [이동] 클릭 시)

```
┌─────────────────────────┐
│  이동할 폴더 선택         │
│                         │
│  ○ 우리집 일상           │  ← 현재 폴더 제외한 폴더 목록
│  ○ 데이트               │
│  ○ 기념일               │
│                         │
│  ┌───────────────────┐  │
│  │  이동              │  │  ← primary, 폴더 선택 시 활성화
│  └───────────────────┘  │
│                         │
└─────────────────────────┘
```

- 현재 폴더는 목록에서 제외
- 이동 완료 시 토스트 알림: "3장의 사진을 '데이트'로 이동했어요"
- 선택 모드 자동 해제

### 태그 상세 — 필터 칩 + 슬라이드쇼 버튼

```
┌─────────────────────────┐
│  ← #벚꽃          [▶ 슬라이드쇼] │
├─────────────────────────┤
│  [ 전체 | 나 | 연인 ]    │  ← 업로더 필터 칩
├─────────────────────────┤
│                           │
│  ┌─────┐ ┌─────┐        │
│  │ 🧑‍  │ │     │        │  ← 2열 그리드 + 아바타 뱃지
│  └─────┘ └─────┘        │
│  ...                     │
│                           │
└─────────────────────────┘
```

- 폴더 상세와 동일한 구조 (필터 칩 + 그리드 + 슬라이드쇼)
- 선택 모드는 태그 상세에는 없음 (폴더 상세에서만)

## 기술 명세

### 업로더 필터링

```ts
// 클라이언트 사이드 필터링
// 서버에서 가져온 사진 목록에서 uploadedBy 기준으로 필터
photos.filter((p) => p.uploadedBy === userId);
```

- "전체": 필터 없음
- "나": `uploadedBy === currentUser.uid`
- "연인": `uploadedBy !== currentUser.uid`

### 일괄 이동

```ts
// Firestore batch write로 여러 사진의 folderId를 한번에 업데이트
const batch = writeBatch(db);
selectedPhotoIds.forEach((photoId) => {
  const ref = doc(db, `couples/${coupleId}/photos/${photoId}`);
  batch.update(ref, { folderId: targetFolderId });
});
await batch.commit();
```

- 이동 완료 후 현재 폴더와 대상 폴더의 사진 목록 invalidate
