# 페이지 명세: 연간 예산 계획 (/cashbook/plan/annual)

## 목적

신혼부부가 한 해의 수입·지출·재테크·Flex 예산을 체계적으로 계획하고, 각 항목의 월별 분배까지 설정할 수 있게 한다. 전년도 데이터가 있으면 이를 기반으로 자동 제안한다.

---

## 사용자 스토리

- 커플로서, 올해 총 수입과 지출을 미리 계획하고 싶다.
- 커플로서, 전년도 실적을 바탕으로 올해 예산을 빠르게 세우고 싶다.
- 커플로서, 수입에서 지출을 빼고 남는 금액으로 재테크와 Flex를 배분하고 싶다.
- 커플로서, 특정 지출이 몇 월에 발생하는지 미리 지정해두고 싶다.

### 3.1.1 연간 수입 계획 탭

정기 수입은 고정으로 자동 반영되고, 비정기 수입만 사용자가 입력한다.

```
│  ── 정기 수입 (자동) ──────────  │
│  ┌───────────────────────────┐  │
│  │ 💰 정기급여  3,500,000/월  │  │  ← 고정값, 수정 가능
│  │    × 12개월 = 42,000,000  │  │
│  │ 🎁 상여     2,000,000/년  │  │
│  │    합계: 44,000,000       │  │
│  └───────────────────────────┘  │
│                                 │
│  ── 비정기 수입 (입력) ────────  │
│  ┌───────────────────────────┐  │
│  │ 🏆 인센티브   5,000,000   │  │  ← 연간 예상 금액 입력
│  │ 💼 부업      3,000,000   │  │
│  │ 🔄 중고거래   2,000,000   │  │
│  │    합계: 10,000,000       │  │
│  └───────────────────────────┘  │
│                                 │
│  ┌───────────────────────────┐  │
│  │  총 연간 수입: 54,000,000  │  │
│  └───────────────────────────┘  │
```

#### 전년도 데이터 연동

전년도 데이터가 있는 경우:

```
│  ┌───────────────────────────┐  │
│  │ 💡 전년도 기반 제안        │  │
│  │                           │  │
│  │ 2025년 비정기 수입:        │  │
│  │ 인센티브 4,500,000        │  │
│  │                           │  │
│  │ 올해 제안 (+5%):          │  │
│  │ 인센티브 4,725,000        │  │
│  │                           │  │
│  │ 증감률 조절: [-] 5% [+]   │  │  ← 슬라이더 또는 ± 버튼
│  │                           │  │
│  │ [제안 적용] [직접 입력]    │  │
│  └───────────────────────────┘  │
```

- 증감률 기본값: **+5%**
- ± 버튼으로 1% 단위 조절 가능
- "제안 적용" 클릭 시 계산된 금액으로 자동 입력
- "직접 입력" 클릭 시 금액 필드에 포커스

#### 지출 시기 설정

비월별 지출(명절 용돈, 보험 연납 등)은 발생 월을 지정할 수 있다.

```
│  지출 시기 설정              │
│  ┌─────────────────────────┐│
│  │ 🧧 명절 용돈 1,000,000   ││
│  │                         ││
│  │  1  2  3  4  5  6      ││
│  │ [●] ○  ○  ○  ○  ○      ││  ← 월 선택 (다중)
│  │  7  8  9  10 11 12     ││
│  │  ○  ○ [●]  ○  ○  ○      ││
│  │                         ││
│  │ 선택: 1월(500,000),      ││  ← 균등 분배
│  │       9월(500,000)       ││
│  └─────────────────────────┘│
```

- 선택한 월에 금액을 균등 분배 (수동 비율 조절도 가능)
- 설정된 시기는 월별 대시보드에 자동 반영

### 3.1.3 연간 재테크 계획 탭

```
│  ┌───────────────────────────┐│
│  │ 📊 재테크 가능 금액        ││
│  │                           ││
│  │ 연간 수입:   54,000,000   ││
│  │ 연간 지출:  -36,000,000   ││
│  │ ─────────────────────────││
│  │ 가용 금액:   18,000,000   ││
│  │                           ││
│  │ 목표 수익률: [3]%         ││  ← 입력 가능
│  │ 목표 금액:   18,540,000   ││  ← 가용 × (1 + 수익률%)
│  └───────────────────────────┘│
│                               │
│  ── 배분 계획 ────────────── │
│  ┌───────────────────────────┐│
│  │ 🏦 예적금    6,000,000    ││  ← 금액 직접 입력
│  │ 💳 입출금    2,000,000    ││
│  │ 📈 국내주식  4,000,000    ││
│  │ 🌍 해외주식  4,000,000    ││
│  │ 📄 채권     2,000,000    ││
│  │ ─────────────────────────││
│  │ 배분 합계:  18,000,000    ││
│  │ 미배분:          0        ││  ← 가용 - 배분 합계
│  └───────────────────────────┘│
```

- **재테크 가능 금액** = 수입 계획 - 지출 계획
- **목표 금액** = 가용 금액 × (1 + 목표 수익률%)
- 배분 합계가 가용 금액을 초과하면 경고 표시
- 미배분 금액이 있으면 Flex 탭에서 사용 가능

## 연간 계획 흐름 요약

```
수입 계획 (3.1.1)
    │
    ├── 정기 수입 (자동/수정 가능)
    └── 비정기 수입 (입력, 전년도 연동 가능)
         │
         ▼
    총 수입 ─────────────────────────┐
                                     │
지출 계획 (3.1.2)                     │
    │                                │
    ├── 고정 지출 (월/연 단위 입력)    │
    ├── 변동 지출-공통                │
    └── 변동 지출-각자                │
         │                           │
         ▼                           │
    총 지출 ─────────────────┐       │
                              ▼       ▼
                        재테크 가능 금액
                        = 수입 - 지출
                              │
                              ▼
                     재테크 계획 (3.1.3)
                     목표 = 가용 × (1+n%)
                     카테고리별 배분
                              │
                              ▼
                        미배분 잔액
                              │
                              ▼
                      Flex 계획 (3.1.4)
                      공동 + 각자 한도 설정
```

## 데이터 모델

```ts
type AnnualPlan = {
  id: string;
  coupleId: string;
  year: number;                      // 예: 2026
  createdBy: string;
  updatedAt: Timestamp;
  createdAt: Timestamp;
};

type AnnualPlanItem = {
  id: string;
  planId: string;                    // AnnualPlan.id 참조
  coupleId: string;
  categoryId: string;                // CashbookCategory.id 참조
  group: CategoryGroup;              // 'income' | 'expense' | 'investment' | 'flex'
  subGroup: CategorySubGroup;
  annualAmount: number;              // 연간 총 예산
  monthlyAmount: number | null;      // 월별 금액 (고정 지출 등)
  targetMonths: number[] | null;     // 지출 시기 [1, 9] (비월별 지출)
  ownerUid: string | null;           // null = 공통, userId = 각자
  updatedAt: Timestamp;
};

type InvestmentPlan = {
  id: string;
  planId: string;                    // AnnualPlan.id 참조
  coupleId: string;
  targetReturnRate: number;          // 목표 수익률 (%, 예: 3)
  totalAvailable: number;            // 재테크 가능 금액 (자동 계산)
  targetAmount: number;              // 목표 금액 (자동 계산)
  updatedAt: Timestamp;
};
```

**Firestore 경로**:
- `couples/{coupleId}/annualPlans/{planId}`
- `couples/{coupleId}/annualPlans/{planId}/items/{itemId}`
- `couples/{coupleId}/annualPlans/{planId}/investmentPlan` (단일 문서)

## 계획 → 월별 연동

연간 계획이 저장되면 월별 예산이 자동 생성된다:

- **월별 금액이 있는 항목**: 매월 동일 금액 배정
- **지출 시기가 지정된 항목**: 해당 월에만 배정 (균등 분배)
- **연간 총액만 있는 항목**: 12로 나눠 균등 배정

이 데이터는 월간 대시보드(`04c-cashbook-monthly.md`)에서 "목표 vs 실제" 비교에 사용된다.
