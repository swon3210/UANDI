# 페이지 명세: 주간 예산 & 지출 (/cashbook/weekly)

## 목적

주 단위로 예산과 실제 지출을 비교하여 일상적인 소비를 관리한다. 월 예산을 주 단위로 나눠 배정하고, 매주 지출 현황을 추적한다.

---

## 사용자 스토리

- 커플로서, 이번 주에 얼마까지 쓸 수 있는지 알고 싶다.
- 커플로서, 지난 주에 초과 지출했으면 이번 주 예산에서 조정하고 싶다.
- 커플로서, 매주 정해진 요일에 지출을 기록하라는 알림을 받고 싶다.

---

## 주(Week) 기준

- **시작**: 월요일
- **종료**: 일요일
- 월의 주 수 계산: 해당 월의 월요일 수 기준
  - 예: 2026년 3월 → 3/2, 3/9, 3/16, 3/23, 3/30 = 5주

> 1일이 월요일이 아닌 경우, 첫 주는 짧은 주(1일~첫 번째 일요일)가 된다.

## 주별 예산 분배 로직

### 기본 분배

```ts
// 월 변동지출 예산에서 고정지출을 제외한 금액을 주 수로 나눔
const monthlyVariableBudget = monthlyBudget - fixedExpenses;
const weeklyBudget = Math.floor(monthlyVariableBudget / numberOfWeeks);
```

- **고정 지출**(월세, 보험 등)은 주별 분배에서 제외
- **변동 지출**(식비, 데이트 등)만 주별로 분배
- 짧은 주(월 시작/끝)는 일수에 비례해 축소 배정

### 이월 로직

```ts
// 전 주에 남은/초과된 금액을 이번 주에 반영
const thisWeekBudget = baseBudget + previousWeekRemaining;
// previousWeekRemaining > 0 → 전주 절약분 이월
// previousWeekRemaining < 0 → 전주 초과분 차감
```

- 이월 적용 여부는 설정에서 on/off 가능 (기본: on)

## 푸시 알림

### 지출 기록 알림

- 설정에서 "기록 요일"을 선택 (기본: 매일 저녁 9시)
- 선택된 요일에 푸시 알림 발송: "오늘 지출을 기록해주세요!"

### 예산 경고 알림

| 조건             | 알림 메시지                        |
| ---------------- | ---------------------------------- |
| 주 예산 80% 소진 | "이번 주 예산이 거의 다 됐어요!"   |

## 진화 버전 (향후 구현)

### 카드 내역 캡처 자동 분류

- 카드 사용 내역 스크린샷을 업로드
- OCR로 금액·가맹점·날짜를 추출
- AI가 카테고리를 자동 추천
- 사용자가 확인 후 일괄 등록

> MVP에서는 수동 입력만 지원. 캡처 기능은 향후 버전에서 구현.
