# 페이지 명세: 사진 갤러리 (종합)

## 목적

커플이 함께 찍은 사진을 **폴더**로 분류하고, **태그**로 횡단 검색하며, 슬라이드쇼로 감상하는 공간.

---

## 라우트 구조

| 경로                                            | 설명                                 |
| ----------------------------------------------- | ------------------------------------ |
| `/photos`                                       | 갤러리 메인 (탭: 전체 / 폴더 / 태그) |
| `/photos/folder/[folderId]`                     | 특정 폴더의 사진 목록                |
| `/photos/tag/[tagName]`                         | 특정 태그의 사진 목록 (폴더 무관)    |
| `/photos/[id]`                                  | 사진 상세                            |
| `/photos/slideshow?source=folder&id=[folderId]` | 슬라이드쇼 (폴더 기준)               |
| `/photos/slideshow?source=tag&id=[tagName]`     | 슬라이드쇼 (태그 기준)               |

---

## 사용자 스토리

- 커플로서, 우리가 함께한 사진들을 **폴더별로 정리**해서 보고 싶다.
- 커플로서, 사진을 올릴 때 **내가 올린 사진**인지 **연인이 올린 사진**인지 구분하고 싶다.
- 커플로서, 여러 폴더에 흩어진 사진을 **태그 기준으로 모아**보고 싶다.
- 커플로서, 폴더나 태그 단위로 사진을 **슬라이드쇼**로 감상하고 싶다.
- 커플로서, 새 사진을 쉽게 업로드하고 캡션·태그를 달고 싶다.
