## UI 구성

### 슬라이드쇼 (/photos/slideshow)

폴더 또는 태그 상세 페이지에서 [▶] 버튼으로 진입.
**전체화면(100vw × 100vh)** 풀스크린 뷰. 배경 검정.

#### UI 상태: Active (인터랙션 직후)

터치, 마우스 클릭/이동 등 인터랙션 발생 시 오버레이 UI가 표시된다.

```
┌──────────────────────────────────┐
│  ✕                        [Aa] │  ← 상단 오버레이: 닫기 + 캡션 토글
│                                 │
│                                 │
│         (사진 전체화면)          │  ← object-contain, 화면 전체
│                                 │
│                                 │
│  📁 제주도 여행                  │  ← 하단 오버레이: 폴더 (클릭 → 이동)
│  #벚꽃 #바다                     │  ← 태그 (클릭 → 이동)
│  3 / 12                          │  ← 위치 표시
│  이 날 정말 행복했어 ☺           │  ← 캡션 (토글 ON일 때만)
└──────────────────────────────────┘
```

#### UI 상태: Idle (5초 무조작)

인터랙션 없이 **5초** 경과 시 오버레이 UI가 fade out되고 사진만 남는다.
다시 터치/마우스 이동 시 오버레이가 fade in으로 복귀.

```
┌──────────────────────────────────┐
│                                 │
│                                 │
│         (사진만 표시)            │
│                                 │
│                                 │
└──────────────────────────────────┘
```

#### 사진 전환 방식

- **넘기기 버튼 없음** — 별도의 화살표/버튼 UI를 표시하지 않는다
- **좌우 영역 탭**: 화면 좌측 절반 클릭 → 이전 사진, 우측 절반 클릭 → 다음 사진
- **좌우 스와이프**: 터치 디바이스에서 스와이프로도 전환 가능
- **키보드 화살표**: `←` 이전 사진, `→` 다음 사진
- 첫 번째 사진에서 이전 / 마지막 사진에서 다음 → 동작 없음 (순환하지 않음)

#### 슬라이드쇼 기능 상세

| 기능 | 설명 |
| --- | --- |
| **사진 전환** | 좌우 영역 탭 + 좌우 스와이프 + 키보드 ←→ (버튼 UI 없음) |
| **자동 숨김** | 5초 무조작 시 오버레이 UI fade out, 인터랙션 시 fade in |
| **캡션 토글** | 우상단 [Aa] 버튼으로 캡션 표시/숨김 (토글 상태는 세션 내 유지) |
| **위치 표시** | "3 / 12" 형태로 현재 순번 / 전체 사진 수 |
| **폴더 표시** | 현재 사진이 속한 폴더명 표시 (클릭 → 폴더 상세 이동) |
| **태그 표시** | 현재 사진의 태그 표시 (클릭 → 태그 상세 이동) |
| **닫기** | ✕ 버튼 |

> 슬라이드쇼에서 폴더/태그 클릭 시 슬라이드쇼를 빠져나와 해당 컬렉션 페이지로 이동한다.

## 기술 명세

### 슬라이드쇼 구현

- **전체화면**: `fixed inset-0 z-50 bg-black`
- **사진 전환**:
  - 좌우 스와이프: `touch` 이벤트 기반 (또는 경량 swipe 라이브러리)
  - 좌우 영역 탭: 화면 좌측 절반 클릭 → 이전, 우측 절반 클릭 → 다음
  - 키보드: `ArrowLeft` → 이전, `ArrowRight` → 다음
  - 넘기기 버튼 UI 없음
- **오버레이 자동 숨김**:
  - 5초 무조작 시 오버레이(상단 바 + 하단 정보) fade out (`opacity-0 transition-opacity duration-300`)
  - `touchstart`, `mousemove`, `click` 이벤트 시 fade in + 타이머 리셋
  - `useRef`로 타이머 ID 관리
- **상태**: 현재 인덱스, 캡션 표시 여부, 오버레이 표시 여부 (`useState`)
- **사진 목록**: 진입 시 쿼리 파라미터(`source`, `id`)로 해당 폴더/태그 사진 전체 로드
- **텍스트**: `text-white`, 오버레이 배경: `bg-gradient-to-b from-black/50` (상단), `bg-gradient-to-t from-black/60` (하단)

### 이미지 표시 최적화

- 슬라이드쇼: `sizes="100vw"`, `priority` 적용 (현재 + 인접 이미지)
