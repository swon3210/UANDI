# 페이지 명세: 카테고리 설정 (/cashbook/categories)

## 목적

수입·지출·재테크·Flex의 카테고리를 체계적으로 관리하고, 각 카테고리에 아이콘·색상을 부여해 직관적으로 구분할 수 있게 한다.

---

## 카테고리 체계

### 2.1 수입 카테고리

| 구분            | 카테고리 | 기본 아이콘 | 기본 색상        | 설명                        |
| --------------- | -------- | ----------- | ---------------- | --------------------------- |
| **정기 수입**   | 정기급여 | 💰          | `--income`       | 매달 고정으로 들어오는 급여 |
|                 | 상여     | 🎁          | `--income`       | 분기/반기/연말 상여금       |
| **비정기 수입** | 인센티브 | 🏆          | `--income-light` | 성과 기반 인센티브          |
|                 | 부업     | 💼          | `--income-light` | 부업·프리랜서 수입          |
|                 | 중고거래 | 🔄          | `--income-light` | 중고물품 판매 수익          |

### 2.2 지출 카테고리

#### 고정 지출 (매달 일정하게 나가는 비용)

| 카테고리  | 기본 아이콘 | 설명                       |
| --------- | ----------- | -------------------------- |
| 월세      | 🏠          | 월세·전세 이자             |
| 금융비용  | 🏦          | 대출 이자, 카드 연회비 등  |
| 보험      | 🛡️          | 건강보험, 자동차보험 등    |
| 공과금    | 💡          | 전기·수도·가스·인터넷      |
| 명절 용돈 | 🧧          | 명절·경조사 관련 고정 지출 |

#### 변동 지출 — 공통 (커플 공동 지출)

| 카테고리 | 기본 아이콘 | 설명                           |
| -------- | ----------- | ------------------------------ |
| 식비     | 🍚          | 장보기, 외식, 배달 (공동)      |
| 소모품   | 🧴          | 생활용품, 소모품               |
| 패션     | 👗          | 의류·신발·액세서리 (공동 구매) |
| 데이트   | 💕          | 데이트 비용                    |
| 건강     | 💪          | 헬스장, 건강식품 등 (공동)     |

#### 변동 지출 — 각자 (개인별 지출)

| 카테고리 | 기본 아이콘 | 설명                 |
| -------- | ----------- | -------------------- |
| 식비     | 🍱          | 개인 점심, 간식      |
| 사회생활 | 🤝          | 회식, 모임           |
| 친구     | 🎉          | 친구 약속, 경조사    |
| 병원     | 🏥          | 개인 병원비          |
| 교통     | 🚌          | 출퇴근, 교통비       |
| 가족     | 👨‍👩‍👧          | 친정·시댁 용돈, 선물 |
| 자기계발 | 📚          | 도서, 강의, 자격증   |

> **공통 vs 각자 구분**: 항목 추가 시 "공통" 또는 "나의 지출"을 선택한다. 각자 지출은 해당 유저에게만 귀속되며, 상대방도 조회는 가능하다.

### 2.3 재테크 카테고리

| 구분          | 카테고리 | 기본 아이콘 | 설명               |
| ------------- | -------- | ----------- | ------------------ |
| **현금 보유** | 예적금   | 🏦          | 정기예금, 적금     |
|               | 입출금   | 💳          | 입출금 통장 잔고   |
| **투자**      | 국내주식 | 📈          | 국내 주식 투자     |
|               | 해외주식 | 🌍          | 해외 주식·ETF 투자 |
|               | 채권     | 📄          | 국채·회사채 등     |

> **재테크 가능 금액** = `총 수입 - 총 지출`로 자동 계산되어 표시.

### 2.4 Flex 카테고리

| 구분          | 카테고리 | 기본 아이콘 | 설명                 |
| ------------- | -------- | ----------- | -------------------- |
| **공동 Flex** | 여행     | ✈️          | 여행 경비            |
|               | 여가     | 🎬          | 영화, 공연, 문화생활 |
| **각자 Flex** | 소비     | 🛍️          | 각자 자유 소비       |
|               | 여가     | 🎮          | 각자 취미·여가       |

> Flex는 예산 한도 내에서 자유롭게 쓸 수 있는 금액. 재테크 배분 후 남은 금액에서 설정.

## 카테고리 데이터 모델

> 상세 타입은 `03-domain-models.md`에 정의

```ts
type CategoryGroup = 'income' | 'expense' | 'investment' | 'flex';

type CategorySubGroup =
  // 수입
  | 'regular_income' // 정기 수입
  | 'irregular_income' // 비정기 수입
  // 지출
  | 'fixed_expense' // 고정 지출
  | 'variable_common' // 변동 지출 (공통)
  | 'variable_personal' // 변동 지출 (각자)
  // 재테크
  | 'cash_holding' // 현금 보유
  | 'investment' // 투자
  // Flex
  | 'joint_flex' // 공동 Flex
  | 'personal_flex'; // 각자 Flex

type CashbookCategory = {
  id: string;
  coupleId: string;
  group: CategoryGroup;
  subGroup: CategorySubGroup;
  name: string; // 카테고리 이름
  icon: string; // 이모지 아이콘
  color: string; // HEX 색상 코드
  isDefault: boolean; // 기본 프리셋 여부
  sortOrder: number; // 정렬 순서
  createdAt: Timestamp;
};
```

**Firestore 경로**: `couples/{coupleId}/cashbookCategories/{categoryId}`

## 기본 프리셋 초기화

커플 생성 시 위 표의 카테고리를 기본값으로 자동 생성한다. 사용자는 이후 추가/편집/삭제/순서 변경이 가능하다.

- 기본 프리셋은 `isDefault: true`로 생성
- 기본 프리셋도 삭제 가능하되, 해당 카테고리를 사용 중인 항목이 있으면 확인 다이얼로그 표시
- 카테고리 삭제 시 기존 항목의 카테고리는 유지 (orphaned 허용)

## 항목 추가 시 카테고리 선택 UX 변경

기존 `04-cashbook.md`의 항목 추가 Bottom Sheet에서 카테고리 선택 부분이 변경된다:

1. **type 탭 확장**: `지출 | 수입` → `지출 | 수입 | 재테크 | Flex`
2. **서브그룹 표시**: 탭 선택 후 서브그룹별로 칩을 그룹핑해서 표시
3. **지출 탭 추가 옵션**: `공통 | 나의 지출` 토글 (변동 지출인 경우)

```
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ │
│  │ 지출 │ │ 수입 │ │재테크│ │ Flex │ │
│  └──────┘ └──────┘ └──────┘ └──────┘ │
│                                       │
│  ── 고정 지출 ──                       │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐           │
│  │🏠│ │🏦│ │🛡│ │💡│ │🧧│           │
│  │월세│ │금융│ │보험│ │공과금│ │명절│     │
│  └──┘ └──┘ └──┘ └──┘ └──┘           │
│                                       │
│  ── 변동 지출 (공통) ──               │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐           │
│  │🍚│ │🧴│ │👗│ │💕│ │💪│           │
│  └──┘ └──┘ └──┘ └──┘ └──┘           │
│                                       │
│  [공통 ←→ 나의 지출]  ← 토글          │
```
