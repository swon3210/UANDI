# 페이지 명세: 투자 placeholder (/outer/investment)

## 목적

v1에는 투자 기능 본 구현이 들어가지 않는다. 사이드바 재테크 섹션의 "투자" 항목이 비어 보이지 않도록 **준비 중 안내 페이지**만 둔다. v1.1에서 본 구현으로 대체한다.

---

## v1 UI

```
┌───────────────────────────────┐
│ [💼 재테크 ▾]      ⚙️  [👤]    │
├───────────────────────────────┤
│                               │
│           📈                  │
│                               │
│        투자 관리              │
│       v1.1에서 만나요         │
│                               │
│   보유 종목과 평가액을        │
│   직접 기록하고               │
│   재테크 대시보드에서           │
│   합산해 볼 수 있어요.        │
│                               │
│   [환테크로 가기] [홈으로]    │
│                               │
├───────────────────────────────┤
│ [홈] [환테크] [투자*] [적금] │
└───────────────────────────────┘
```

- 중앙 정렬, `EmptyState` 패턴 활용 (`@uandi/ui`)
- 두 CTA 버튼:
  - "환테크로 가기" → `/outer/forex`
  - "홈으로" → `/outer`

---

## 동작 규칙

- 로그인·커플 연결 가드는 AppShell이 처리
- 별도 데이터 요청 없음
- 페이지 메타 description은 재테크 공간 소개 그대로 사용

---

## 향후 (v1.1)

v1.1 본 구현 시 이 문서를 다음으로 확장한다:

- 보유 종목 목록 (`couples/{coupleId}/sideHustles/{uid}/positions/*`)
- 종목 추가/수정/삭제
- 매입가·현재가 수동 입력 → 평가손익 계산
- `meta/outerSummary.byUser[uid].investment` 갱신

상세 도메인 타입은 `docs/03-domain-models.md`의 `Position` 참고.
