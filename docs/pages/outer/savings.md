# 페이지 명세: 적금 placeholder (/outer/savings)

## 목적

v1에는 적금 기능 본 구현이 들어가지 않는다. 재테크 BottomNav의 "적금" 탭이 비어 보이지 않도록 **준비 중 안내 페이지**만 둔다. v1.1에서 본 구현으로 대체한다.

---

## v1 UI

```
┌───────────────────────────────┐
│ [💼 재테크 ▾]      ⚙️  [👤]    │
├───────────────────────────────┤
│                               │
│           🐖                  │
│                               │
│        적금 트래커            │
│       v1.1에서 만나요         │
│                               │
│   적금 만기일과 이율을        │
│   한눈에 추적하고             │
│   재테크 대시보드에서           │
│   합산해 볼 수 있어요.        │
│                               │
│   [환테크로 가기] [홈으로]    │
│                               │
├───────────────────────────────┤
│ [홈] [환테크] [투자] [적금*] │
└───────────────────────────────┘
```

- 중앙 정렬, `EmptyState` 패턴 활용 (`@uandi/ui`)
- 두 CTA 버튼:
  - "환테크로 가기" → `/outer/forex`
  - "홈으로" → `/outer`

---

## 동작 규칙

- 로그인·커플 연결 가드는 AppShell이 처리
- 별도 데이터 요청 없음

---

## 향후 (v1.1)

v1.1 본 구현 시 이 문서를 다음으로 확장한다:

- 적금 계좌 목록 (`couples/{coupleId}/sideHustles/{uid}/savings/*`)
- 계좌 추가/수정/삭제 (은행, 원금, 월 납입액, 이율, 만기일)
- 만기일 D-day 표시, 만기 임박 카드 정렬
- `meta/outerSummary.byUser[uid].savings` 갱신

상세 도메인 타입은 `docs/03-domain-models.md`의 `SavingsAccount` 참고.
