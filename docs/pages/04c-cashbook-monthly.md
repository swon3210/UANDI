# 페이지 명세: 월간 대시보드 (/cashbook/monthly)

## 목적

월별 수입·지출·재테크의 **목표 vs 실제**를 한눈에 비교하고, 현재 재정 상태를 색상으로 직관적으로 파악할 수 있게 한다.

---

## 사용자 스토리

- 커플로서, 이번 달 예산 대비 실제 지출이 어느 정도인지 바로 알고 싶다.
- 커플로서, 초과 지출 위험이 있으면 시각적 경고를 받고 싶다.
- 커플로서, 정기 수입은 자동으로 반영되고 비정기 수입만 입력하고 싶다.
- 커플로서, 투자 매수·매도 내역을 기록하고 싶다.

---

## 상태 색상 체계

| 상태 | 색상                  | 조건                    | 의미           |
| ---- | --------------------- | ----------------------- | -------------- |
| 안정 | 🟢 초록 (`--income`)  | 여유분 ≥ 예산의 20%     | 예산 내 안정적 |
| 경고 | 🟡 노랑 (`--warning`) | 0 < 여유분 < 예산의 20% | 예산 소진 임박 |
| 긴급 | 🔴 빨강 (`--expense`) | 여유분 ≤ 0 (마이너스)   | 예산 초과      |

> **여유분** = 월 예산 - 월 실제 지출

### 지출 탭 상세

카테고리별 + 주별로 예산 대비 지출을 표시한다.

```
│  ── 카테고리별 지출 ────────  │
│  ┌───────────────────────┐  │
│  │ 🍚 식비                │  │
│  │ ██████░░░░  65%        │  │  ← 390,000 / 600,000
│  │ 🟢 여유: 210,000       │  │
│  ├───────────────────────┤  │
│  │ 🏠 월세                │  │
│  │ ██████████ 100%        │  │  ← 800,000 / 800,000
│  │ ⚪ 완료                │  │
│  ├───────────────────────┤  │
│  │ 🤝 사회생활 (나)       │  │
│  │ ████████░░  85%        │  │  ← 170,000 / 200,000
│  │ 🟡 주의: 30,000        │  │
│  └───────────────────────┘  │
│                             │
│  ── 주별 지출 추이 ────────  │
│  ┌───────────────────────┐  │
│  │  주차  │ 예산    │ 실제    │ 상태 │
│  │  1주차 │ 750,000 │ 680,000 │ 🟢  │
│  │  2주차 │ 750,000 │ 820,000 │ 🔴  │
│  │  3주차 │ 750,000 │ 360,000 │ 🟢  │  ← 진행 중
│  │  4주차 │ 750,000 │   -     │ ⚪  │  ← 미래
│  └───────────────────────┘  │
```

- 주별 예산 = 월 변동지출 예산 ÷ 해당 월의 주 수
- 고정 지출은 주별 분배에서 제외 (월 단위로만 표시)

### 수입 탭 상세

```
│  ── 정기 수입 (자동 반영) ──  │
│  ┌───────────────────────┐  │
│  │ 💰 정기급여 3,500,000  │  │  ← 계획에서 자동 반영
│  │    상태: ✅ 반영됨      │  │
│  │ 🎁 상여       -        │  │  ← 이번 달 해당 없음
│  └───────────────────────┘  │
│                             │
│  ── 비정기 수입 (입력) ────  │
│  ┌───────────────────────┐  │
│  │ 🏆 인센티브     -      │  │  ← 미입력
│  │ 💼 부업    250,000     │  │  ← 사용자 입력
│  │ [+ 비정기 수입 추가]    │  │
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │ 계획: 4,500,000        │  │
│  │ 실제: 3,750,000        │  │
│  │ 달성률: 83%            │  │
│  └───────────────────────┘  │
```

- **정기 수입**: 연간 계획의 월별 금액이 자동 반영됨
  - "반영됨" = 해당 월의 실제 cashbookEntry가 존재
  - 아직 입력 안 된 정기 수입은 "예정"으로 표시하되 실제 합계에는 미포함
- **비정기 수입**: 사용자가 직접 입력 (기존 가계부 항목 추가와 동일)

## 월별 예산 자동 분배 로직

```
월 예산 = (연간 계획 아이템별)
  - monthlyAmount가 있으면 → 그대로 사용
  - targetMonths가 있으면 → 해당 월에만 배정 (annualAmount ÷ targetMonths.length)
  - 둘 다 없으면 → annualAmount ÷ 12
```

## 데이터 모델 (추가)

기존 `CashbookEntry` 타입을 확장한다:

```ts
// CashbookEntry.type 확장
type EntryType = 'income' | 'expense' | 'investment' | 'flex';

// 투자 항목 추가 필드
type InvestmentEntry = CashbookEntry & {
  type: 'investment';
  transactionType: 'buy' | 'sell'; // 매수 / 매도
};

// 월별 예산 스냅샷 (연간 계획에서 자동 생성)
type MonthlyBudget = {
  id: string;
  coupleId: string;
  planId: string; // AnnualPlan.id
  year: number;
  month: number; // 1~12
  items: MonthlyBudgetItem[];
  createdAt: Timestamp;
  updatedAt: Timestamp;
};

type MonthlyBudgetItem = {
  categoryId: string;
  group: CategoryGroup;
  subGroup: CategorySubGroup;
  budgetAmount: number; // 이번 달 예산
  ownerUid: string | null; // null = 공통
};

// 현금 보유 잔고 (월말 스냅샷)
type CashBalance = {
  id: string;
  coupleId: string;
  categoryId: string; // 예적금 or 입출금 카테고리
  year: number;
  month: number;
  balance: number; // 잔고
  updatedAt: Timestamp;
};
```

**Firestore 경로**:

- `couples/{coupleId}/monthlyBudgets/{year}-{month}`
- `couples/{coupleId}/cashBalances/{balanceId}`
