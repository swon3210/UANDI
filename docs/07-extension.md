# 크롬 익스텐션

UANDI 크롬 익스텐션은 현재 보고 있는 웹페이지의 투명도를 조절하는 도구입니다.
가계부 기능은 웹앱에서만 제공합니다.

---

## 주요 기능

### 웹페이지 투명도 조절

현재 탭의 페이지 전체 투명도를 슬라이더로 조절합니다.

- 슬라이더 값에 따라 페이지의 `document.documentElement` opacity가 즉시 변경됩니다
- "초기화" 버튼으로 100%로 되돌릴 수 있습니다
- 슬라이더 값은 브라우저 localStorage에 저장되어 다음에 팝업을 열 때 동일한 값으로 적용됩니다
- 페이지 자체의 opacity는 새로고침 시 1로 초기화됩니다 (Chrome scripting 특성)
- `chrome://` 같은 시스템 페이지에서는 동작하지 않으며 오류 없이 무시됩니다

## 다운로드 및 설치

### 1단계: 익스텐션 파일 다운로드

아래 링크에서 익스텐션 파일(`dist.zip`)을 다운로드합니다.

👉 [UANDI 크롬 익스텐션 다운로드](https://firebasestorage.googleapis.com/v0/b/uandi-55ee4.firebasestorage.app/o/extension%2Fdist.zip?alt=media&token=8e683017-68b3-446a-81d2-333045d2bbb5)

### 2단계: 압축 해제

다운로드한 `dist.zip` 파일의 압축을 해제합니다.
압축을 풀면 `dist` 폴더가 생성됩니다.

### 3단계: 크롬 확장 프로그램 페이지 열기

Chrome 브라우저의 주소창에 아래 주소를 입력합니다.

```
chrome://extensions
```

또는 **Chrome 메뉴(⋮) → 확장 프로그램 → 확장 프로그램 관리**로 이동합니다.

### 4단계: 개발자 모드 활성화

확장 프로그램 페이지 우측 상단의 **"개발자 모드"** 토글을 켭니다.

### 5단계: 익스텐션 로드

**"압축 해제된 확장 프로그램을 로드합니다"** 버튼을 클릭한 뒤,
2단계에서 압축 해제한 `dist` 폴더를 선택합니다.

### 6단계: 설치 완료

UANDI 도구 아이콘이 Chrome 툴바에 나타나면 설치가 완료된 것입니다.
아이콘을 클릭하면 팝업이 열리고, 별도 로그인 없이 바로 사용할 수 있습니다.
