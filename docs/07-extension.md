# 크롬 익스텐션

UANDI 크롬 익스텐션은 브라우저에서 바로 가계부 내역을 입력하고 관리할 수 있는 도구입니다.
웹앱을 열지 않고도 빠르게 수입/지출을 기록할 수 있습니다.

---

## 주요 기능

### AI 자연어 입력

자연어로 가계부 내역을 입력할 수 있습니다.

- **입력 예시**: "점심 김치찌개 9000원", "카페 아메리카노 4500원"
- AI가 자동으로 금액, 카테고리, 날짜를 파싱합니다
- 헤더의 ✨ 버튼을 눌러 AI 입력 모드를 활성화합니다

### 수동 입력 폼

직접 항목을 선택해서 내역을 등록할 수 있습니다.

- 수입/지출 구분
- 카테고리 선택
- 금액, 메모, 날짜 입력
- 헤더의 + 버튼을 눌러 수동 입력 폼을 엽니다

### 월간 대시보드

현재 월의 가계부 요약을 한눈에 확인할 수 있습니다.

- 총 수입 / 총 지출 / 잔액 표시
- 이전/다음 월 이동
- 날짜별 내역 그룹 표시

### 내역 수정 및 삭제

등록된 가계부 내역을 수정하거나 삭제할 수 있습니다.

- 내역을 탭하면 수정 폼이 열립니다
- 수정 폼에서 삭제도 가능합니다

### 실시간 동기화

Firebase Firestore를 통해 커플 계정 간 실시간으로 데이터가 동기화됩니다.
웹앱에서 입력한 내역도 익스텐션에서 바로 확인할 수 있습니다.

## 다운로드 및 설치

### 1단계: 익스텐션 파일 다운로드

아래 링크에서 익스텐션 파일(`dist.zip`)을 다운로드합니다.

👉 [UANDI 크롬 익스텐션 다운로드](https://firebasestorage.googleapis.com/v0/b/uandi-55ee4.firebasestorage.app/o/extension%2Fdist.zip?alt=media&token=8e683017-68b3-446a-81d2-333045d2bbb5)

### 2단계: 압축 해제

다운로드한 `dist.zip` 파일의 압축을 해제합니다.
압축을 풀면 `dist` 폴더가 생성됩니다.

### 3단계: 크롬 확장 프로그램 페이지 열기

Chrome 브라우저의 주소창에 아래 주소를 입력합니다.

```
chrome://extensions
```

또는 **Chrome 메뉴(⋮) → 확장 프로그램 → 확장 프로그램 관리**로 이동합니다.

### 4단계: 개발자 모드 활성화

확장 프로그램 페이지 우측 상단의 **"개발자 모드"** 토글을 켭니다.

### 5단계: 익스텐션 로드

**"압축 해제된 확장 프로그램을 로드합니다"** 버튼을 클릭한 뒤,
2단계에서 압축 해제한 `dist` 폴더를 선택합니다.

### 6단계: 설치 완료

UANDI 가계부 아이콘이 Chrome 툴바에 나타나면 설치가 완료된 것입니다.
아이콘을 클릭하면 팝업이 열리고, Google 로그인 후 바로 사용할 수 있습니다.

> **참고**: 익스텐션을 사용하려면 먼저 UANDI 웹앱에서 커플 연결이 완료되어 있어야 합니다.
