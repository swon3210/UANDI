# UANDI — 프로젝트 개요

## 비전

신혼부부가 함께 만들어가는 일상을 기록하고 관리하는 프라이빗 공간.
둘만의 사진을 보관하고, 돈 관리를 함께 해나갈 수 있는 서비스.

## 핵심 사용자

- **대상**: 신혼부부 (커플 단위 사용)
- **사용 맥락**: 모바일 우선 / 가끔 데스크탑
- **핵심 가치**: 간결함, 프라이버시, 둘만의 공간

---

## 기능 범위 (MVP)

| 영역     | 기능                            | 우선순위 |
| -------- | ------------------------------- | -------- |
| 인증     | 구글 소셜 로그인                | P0       |
| 인증     | 커플 연결 (초대 코드)           | P0       |
| 사진     | 사진 업로드                     | P0       |
| 사진     | 사진 갤러리 (그리드)            | P0       |
| 사진     | 사진 상세 감상                  | P0       |
| 가계부   | 수입/지출 내역 추가             | P0       |
| 가계부   | 월별 내역 조회                  | P0       |
| 가계부   | 월별 요약 (수입/지출/잔액)      | P0       |
| 사진     | 앨범(폴더) 분류                 | P1       |
| 가계부   | 카테고리별 통계                 | P1       |
| 대시보드 | 최근 사진 / 이번 달 가계부 요약 | P1       |

> **P0**: 첫 배포에 포함 / **P1**: 이후 단계에서 추가

## 페이지 구성

```
/                   → 랜딩 (비로그인) / 대시보드 (로그인)
/onboarding         → 최초 가입 + 커플 연결
/photos             → 사진 갤러리
/photos/[id]        → 사진 상세
/cashbook           → 가계부
```
